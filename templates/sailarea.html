ORC Mainsail Area Calculator

This tool uses the official ORC formula to estimate the mainsail area:

Area = (P ÷ 8) × (E + 2·MQW + 2·MHW + 1.5·MTW + MUW + 0.5·MHB)
where:
- P = luff length (mast height)
- E = foot length (boom length)
- MQW = ¼ height sail width
- MHW = ½ height sail width
- MTW = ¾ height sail width
- MUW = ⁷⁄₈ height sail width
- MHB = headboard width
